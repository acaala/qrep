# qrep - Quick Replace

## A Quick, lightweight cli find and replace tool

### Usage

```bash
qrep "from" "to" "path/to/file.txt" "new/path/name.txt"
```


#### Options

| Flag | Default | Description |
| ---- | ------- | ----------- |
| -s (-speed) | 1 | 1 Fastest (minimal feedback), 2 Fast (some feedback), 3 slower (all feedback) |
